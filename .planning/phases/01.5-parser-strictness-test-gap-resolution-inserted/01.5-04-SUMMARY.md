## What Was Built

### Task 1 — `determinism.parse_twice` via PUBLIC C ABI (commit `5effb61`)

New file `tests/unit/determinism_parse_twice.zig` (144 lines) declaring `test "determinism.parse_twice"`. For each of the 19 showcase files:

1. Read the source bytes via `std.Io.Dir.cwd().readFileAlloc`.
2. **First parse** via the PUBLIC C ABI: `lib.deal_parse(source.ptr, source.len, path.ptr, path.len)` → opaque handle.
3. Pull AST JSON via `lib.deal_ast_json(handle, &ptr_a, &len_a)` (out-param style).
4. `gpa.dupe(u8, ptr_a[0..len_a])` to copy the arena-owned bytes into gpa-owned memory **before** `deal_free` invalidates the pointer. Same for diagnostics JSON.
5. Assert `lib.deal_diagnostics_count(handle_a) == 0` (showcase files are well-formed).
6. `lib.deal_free(handle_a)` — invalidates `ptr_a` / `dptr_a` but the duped buffers survive in gpa.
7. **Second parse** via the same PUBLIC C ABI from scratch (`lib.deal_parse(...)` again).
8. Pull parse-B's AST JSON + diagnostics JSON.
9. Assert byte-equality via `std.mem.eql(u8, json_a, ptr_b[0..len_b])`; on mismatch, print the path and call `expectEqualStrings` so the test harness emits a precise diff.
