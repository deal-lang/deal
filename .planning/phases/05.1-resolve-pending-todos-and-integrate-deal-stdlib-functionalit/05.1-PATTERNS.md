### `src/fmt.zig` — replace `__array__` special-case with `array_literal` branch

**Existing special-case to REPLACE** (`src/fmt.zig:754-768`):
```zig

        // Special case: `__array__(arg1, arg2, ...)` is the parser's internal
        // representation of `[arg1, arg2, ...]` array literals. Emit the canonical
        // `[...]` form so the formatter round-trips the original source syntax.
        if (p.callee.kind == .identifier and
            std.mem.eql(u8, p.callee.payload.identifier.name, "__array__"))
        {

            for (p.args, 0..) |arg, i| {
                if (i > 0) try ctx.write(", ");
                try ctx.writeNode(arg, indent);
            }
            try ctx.write("]");
            return;
        }
        // ... normal call formatting continues
    }
```

**Replacement:** Add a new `writeArrayLiteral` method in `fmt.zig`'s `FormatContext` and add a `writeNode` dispatch arm for `.array_literal`. Remove the `__array__`-name check from `writeCall`.
