| T-05-04 | Information Disclosure | non-deterministic key order leaks ordering into hashed evidence | mitigate | json.dump(sort_keys=True) on output.json + metadata.json (D-18) so the staleness hash is byte-stable |
| T-05-05 | Tampering | third-party dependency supply chain in SDK core | mitigate | D-78 stdlib-only core; grep gate forbids numpy/scipy/jsonschema imports; deal-sim created from scratch (not a registry install) |
</threat_model>
