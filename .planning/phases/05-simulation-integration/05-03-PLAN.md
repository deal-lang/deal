| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-05-07 | Tampering | subprocess injection via `runner` field | mitigate | Split runner into command+args; spawn via std::process::Command::new (CWE-78); NEVER shell-interpolate |
| T-05-08 | Denial of Service | MATLAB absent fails the whole run | mitigate | Graceful-skip + skip.json on ErrorKind::NotFound (D-72); continue, do not Err |
